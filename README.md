# Travel Bucket List Dashboard

Interactive dashboard built with **Next.js 14**, **App Router**, **Tailwind CSS v4**, and **Zustand**.

## Features

- Display travel destinations
- Mark destinations as visited
- Sort (A→Z, Z→A)
- Filter (visited / not visited)
- Progress bar
- Fully responsive UI

## Tech Stack

- Next.js 14 (App Router)
- React Server Components + Client Components
- Tailwind CSS v4
- Zustand state management

## Getting Started

```bash
npm install
npm run dev
```
